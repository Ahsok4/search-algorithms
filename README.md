# AI Search ALgorithms

##### This repository contains one of the Fundamentals and Applications of Artificial Intelligence course projects, which I took in the first semester of the Academic year 2021-2022 at the Ferdowsi University of Mashhad.

##### A*, BDS, BFS, DFS, GBFS, IDA*, IDDFS, and RBFS search algorithms are implemented in this project to solve a problem. Also, a visualizer is implemented to show the graphs, which takes the search algorithms output file as input.
